YFetch is a macOS CLI tool inspired by Neofetch. I built it as a small Bash project to practice shell scripting.

## Features

- Displays the current user
- Displays the host name
- Displays the uptime
- Displays the current shell and its version
- Displays the macOS name
- Displays the CPU
- Displays the total memory
- Displays the GPU
- Displays the terminal emulator
- Displays a local image with `catimg`
